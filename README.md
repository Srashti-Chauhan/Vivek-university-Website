# Vivek University Website

This is a responsive multi-page university website developed using HTML and CSS. It represents a modern educational institution website with structured navigation, sections, and visual design.

## 🔹 Features
- Responsive multi-page website
- Navigation bar with multiple pages
- Hero section with background image
- Information sections (About, Courses, Faculty, Admissions)
- Contact page with form layout
- Clean and user-friendly interface

## 🔹 Pages Included
- Home (index.html)
- About (about.html)
- Courses (courses.html)
- Admissions (admissions.html)
- Faculty (faculty.html)
- Contact (contact.html)

## 🔹 Technologies Used
- HTML5
- CSS3

## 🔹 Project Purpose
This project was created as a college assignment to practice front-end web development and design a real-world university website.
